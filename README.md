# Just QR

A minimalist QR code generator. Enter a URL, get a QR code, download it.

## Tech Stack

- Next.js 15
- React
- Tailwind CSS
- shadcn/ui
- qrcode library

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).
